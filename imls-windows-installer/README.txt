=====
Session Counter is a pilot project to automate the monitoring of wifi usage via low-cost and open source tools.

This software was developed by James Tranovich and Matt Jadud, federal software engineers with 18F, with funding support from 10x and in collaboration with the Institute of Museum and Library Services.

Questions can be directed to matthew dot jadud at gsa dot gov.
